CRUD application blog app designed with express.js and MONGODB 

## Features 
- Bootstrap design
- Seo Sitemap and Robot.txt
- TinyMce editor
- Couresel slides
- Ability to add and edit post with images using Multer plugin
- Featured Post
- Post per Category
- Related Post
- Date formatted using Momentum
